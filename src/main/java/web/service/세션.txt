아파치 톰캣 (웹서버 소프트웨어)

세션 : 아파치 톰캣(웹서버) 저장소
    - HTTP 요청에 따른 *브라우저 마다*의 객체 저장소
    - 주로 로그인정보, 장바구니, 인증, 인가에 사용됨
    - 생명주기: 서버가 꺼지면 사라짐, 세션 초기화, 특정 시간
    - 너무 많이 사용되면 웹서버의 과부하, 적절히 사용
    - HTTP 요청 객체
        HttpServletRequest request;
    - 메소드
        1. request.getSession() : 현재 HTTP 요청의 세션 객체 호출
        2. (세션객체).setAttribute("key", value);
        3. (세션객체).getAttribute("key"):
            - 강제 형변환이 필요하다.
        4. (세션객체).invalidate(): 세션 객체내 모든 속성을 제거
